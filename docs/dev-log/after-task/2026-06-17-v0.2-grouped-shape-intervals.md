# After-task: profile + bootstrap intervals for grouped shape coordinates (v0.2)

## Task goal

Complete the grouped-shape feature: give the per-group shape coordinates
(`low:<g>`, `up:<g>`, `k:<g>`) the same interval types as the rest of the model,
not just Wald. First of the quick feature-completions after the v0.2 run.

## What changed

- `R/profile.R`: `tls_resolve_target()` now resolves the shape families. `low:<g>`
  -> `beta_low[g]` (logit, `plogis`) and `k:<g>` -> `beta_logk[g]` (log, `exp`)
  profile via the existing map-refit machinery; `up:<g>` joins the `up` branch
  (no single coordinate under the nested gap -> delta-method Wald, like the
  scalar `up`). New `tls_shape_index()` helper handles the shared-vs-grouped
  distinction (shapes can be shared even when `CTmax` is grouped, so it counts
  `beta_low` coordinates rather than assuming the `CTmax` group count). The
  grouped-shape profile guard was removed; `tls_up_wald_profile()` is
  parameterised by target name; `low`/`k` were dropped from `scalar_map` (now the
  family branches; `phi` stays).
- `R/bootstrap.R`: removed the grouped-shape guard; `refit_one()` collects the
  per-group shape replicates (`low:<g>` / `up:<g>` / `k:<g>`) via shape-name
  vectors, mirroring `ct_names` / `z_names`. `tls_boot_target()`'s direct-parameter
  branch already maps `low:<g>` etc. to their construction scales.

The scalar shape path (`low`, `k`) is byte-identical (the family branch with one
coordinate reproduces the old `scalar_map` index-1 resolution); existing profile
tests pass unchanged.

## Checks run and exact outcomes

- `devtools::test()` -> `PASS 389 | FAIL 0 | WARN 0 | SKIP 0` (was 383; +6 in
  `test-shape-covariates.R`). All existing profile / bootstrap tests pass
  unchanged.
- `rcmdcheck` (`_R_CHECK_FORCE_SUGGESTS_=false`) -> pending in the check-log.
- Behaviour: a fully grouped 2-group fit gives `confint("k:A", method =
  "profile")` a finite ordered interval (`method = "profile"`); `up:A` a finite
  Wald interval; `low:A` a finite bootstrap interval.

## Tests of the tests

`test-shape-covariates.R` adds: a `k:A` profile interval (finite, ordered,
`method = "profile"`), an `up:A` Wald fallback (finite), and a `low:A` bootstrap
interval (`method = "bootstrap"`).
